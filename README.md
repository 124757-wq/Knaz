# 🎮 DOTA 2 STATS - Сайт статистики игроков

Веб-приложение для просмотра статистики игроков Dota 2 через OpenDota API.

## 📋 Что нужно для запуска:

1. **Node.js** - скачай с https://nodejs.org/ (выбери LTS версию)

## 🚀 Как запустить:

### Вариант 1 (Простой):
1. Запусти файл `START.bat`
2. Открой браузер: http://localhost:3000

### Вариант 2 (Через командную строку):
1. Открой командную строку в папке проекта
2. Выполни: `node server.js`
3. Открой браузер: http://localhost:3000

## 📖 Как пользоваться:

1. Найди свой Steam ID на https://www.opendota.com
2. Введи ID в поле поиска
3. Нажми "Найти игрока"
4. Смотри свою статистику!

## 🎯 Возможности:

✅ Поиск игрока по Steam ID
✅ Общая статистика (винрейт, игры, побеги/поражения)
✅ Топ-6 героев с винрейтом
✅ Последние 10 матчей
✅ Отображение ранга игрока
✅ Красивый дизайн в стиле Dota 2

## 🛠️ Технологии:

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js (HTTP сервер)

## 🔧 Как работает сервер:

Сервер выполняет 2 функции:
1. **Раздает статические файлы** (HTML, CSS, JS)
2. **Проксирует запросы к OpenDota API** (решает проблему CORS)

Когда фронтенд делает запрос к `/api/players/123`, сервер:
- Перехватывает запрос
- Отправляет его к `api.opendota.com/api/players/123`
- Возвращает результат с правильными CORS заголовками
